# Desafio Like Me 

## Forma de ejecutar
    

### Front End

 Desde la consola abrir la carpeta frontend y ejecutar 
 ```
 npm run dev
 ```

### Back End

Crear archivo .env con la siguientes variables de entornos

### 

```
DB_USER=root
DB_HOST=127.0.0.1
DB_DATABASE=likeme
DB_PORT=5432
DB_PASSWORD=*****
```
 Desde la consola abrir la carpeta backend y ejecutar 
```
npm run dev
```
